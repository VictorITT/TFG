Para generar la APK a través de Ionic, es necesario ejecutar el comando;
> ionic capacitor build android --release


Comandos ANGULAR:
Instalar Angular > generar dependencias > instalar toastr para tener toasts en la web > arrancar el servicio 
> npm install -g @angular/cli@latest
> npm install
> npm install ngx-toastr --save
> ng serve

Comandos IONIC:
Instalar Ionic de forma global > generar dependencias > arrancar el servicio> buildear la app
> npm install -g @ionic/cli cordova-res
> npm install
> ng serve
> ionic capacitor build android --release
